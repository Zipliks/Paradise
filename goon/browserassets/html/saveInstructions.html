Инструкция по сохранению логов

Ввиду примитивной природы IE8 файл логов придётся сохранять вручную.
  1. Кликните (ЛКМ) под чертой ниже этой инструкции.
  2. Используйте горячую клавишу «Выделить всё» (Ctrl + A)
  3. Используйте горячую клавишу «Копировать» (Ctrl + C)
  4. Создайте новый текстовый документ (ПКМ по рабочему столу (или в Проводнике) и выберите Создать → Текстовый документ
  5. Откройте созданный документ и вставьте туда скопированный текст (Ctrl + V)
  6. Сохраните документ с расширением .html вместо .txt.
